# Incident Response Plan – Healthcare Environment

## Overview
This project outlines a basic Incident Response Plan (IRP) designed for a small healthcare organization, such as a dental or outpatient clinic. The focus is on preparing for and responding to cybersecurity incidents, particularly ransomware attacks that could impact patient data and operations.

## Objective
To demonstrate an understanding of:
- Incident response lifecycle
- Risk management in healthcare environments
- Protection of sensitive data (PHI)
- Real-world cybersecurity response strategies

## Scenario
A small healthcare practice experiences a ransomware attack that encrypts patient records and disrupts daily operations.

## Incident Response Phases

### 1. Preparation
- Implement endpoint protection and antivirus software
- Enable Multi-Factor Authentication (MFA)
- Perform regular data backups (offline and cloud)
- Train staff on phishing awareness

### 2. Identification
- Detect unusual system behavior (locked files, ransom note)
- Identify affected systems and scope of attack
- Confirm type of attack (ransomware)

### 3. Containment
- Disconnect infected systems from the network
- Disable compromised user accounts
- Prevent spread to other devices

### 4. Eradication
- Remove malware from affected systems
- Patch vulnerabilities exploited during the attack
- Run full system scans

### 5. Recovery
- Restore systems from clean backups
- Monitor for signs of reinfection
- Gradually bring systems back online

### 6. Lessons Learned
- Conduct post-incident review
- Update security policies
- Improve response procedures
- Provide additional staff training

## Key Risks Identified
- Lack of employee cybersecurity awareness
- Weak password policies
- No incident response plan in place
- Infrequent data backups

## Recommendations
- Enforce strong password and MFA policies
- Schedule regular backups and testing
- Conduct security awareness training
- Develop and maintain an updated incident response plan

## Tools & Concepts Used
- Incident Response Framework (NIST-inspired)
- Basic risk assessment methodology
- Cybersecurity best practices for healthcare
